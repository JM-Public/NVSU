# NVSU Project

---
**Authentication**
- สามารถสมัครสมาชิกได้
- มีระบบ Login

**User management**
- เรียกดู Profile ของตัวเอง
- เรียกดู Order history ของตัวเอง

**Product management**
- เรียกดู Product ทั้งหมดที่มีในระบบ
- สามารถดูข้อมูลของแต่ละ Product ได้

**Order management**
- สร้างรายการคำสั่งซื้อได้
- ยกเลิกคำสั่งซื้อ
- ดูรายละเอียดคำสั่งซื้อ
